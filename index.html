Skill

Kemampuan saya sebagai Front-End Developer.

HTML

Hypertext Markup Language adalah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan di peramban internet. Ini dapat dibantu oleh teknologi seperti Cascading Style Sheets dan bahasa skrip lainnya seperti JavaScript, VBScript, dan PHP. Dan saya dapat menjalankan dan memahami tag atau atribute yang dipakai atau dibutuhkan seperti div, span, id, class, dll.

CSS

Cascading Style Sheet merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemrograman. Dan saya sering memakainya tetapi tidak secara langsung, melainkan dengan bantuan framework seperti bootstrap atau TailwindCSS.

Javascript

JavaScript adalah suatu bahasa pemrograman tingkat tinggi dan dinamis. Saya sering menggunakan javascript karena dapat melakukan suatu logic atau interaktif pada user, termasuk website ini menggunakan Javascript.

Bootstrap

Bootstrap adalah kerangka kerja CSS yang sumber terbuka dan bebas untuk merancang situs web dan aplikasi web. Saya sering juga menggunakannya karena di bootstrap ini kamu dapat membuat website sederhana dengan mudah.

Tailwind

Tailwind CSS adalah kerangka kerja CSS yang di dalamnya terdapat sekumpulan utility classes untuk membangun antarmuka kustom dengan cepat. Saya sering menggunakannya karena di tailwind kita dapat merangkai komponen dengan memanggil class yang disediakan tailwind ini.

ReactJS

React adalah pustaka JavaScript front-end sumber terbuka dan gratis untuk membangun antarmuka pengguna berdasarkan komponen UI. React ini merupakan library yang saya sukai karena dia bisa hanya membuat satu komponen dan komponen tersebut dapat digunakan berulang kali.

Git

Git adalah perangkat lunak pengendali versi atau proyek manajemen kode perangkat lunak yang diciptakan oleh Linus Torvalds, yang pada awalnya ditujukan untuk pengembangan kernel Linux. Git ini menurut saya seperti Github hanya saja dia tidak ada tampilan secara UI nya.

Github

GitHub adalah layanan hos web bersama untuk proyek pengembangan perangkat lunak yang menggunakan sistem kendali versi Git dan layanan hosting internet. Github ini sebuah web yang keren karena saya dapat menggunakan nya sebagai tempat menyimpan file dan dapat diatur dengan menggunakan Git.
